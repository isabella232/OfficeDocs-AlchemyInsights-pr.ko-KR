# <a name="using-custom-policy-in-intune"></a>Intune에서 사용자 지정 정책 사용

Intune에는 Android, iOS, macOS 및 Windows 기본 제공되어 있습니다. 사용자 지정 프로필을 만든 다음 Intune을 사용하여 이러한 사용자 지정 설정을 장치에 배포할 수도 있습니다. 사용자 지정 프로필은 Intune에 기본 제공되지 않은 장치 설정 및 기능을 추가하도록 디자인되었습니다.

[Intune에서 Android 장치에 대한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-android)

[Intune에서 Android Enterprise 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Intune에서 iOS 장치에 대한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-ios)

[Intune에서 macOS 장치에 대한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-macos)

[Intune에서 Windows 10 장치에 대한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Intune에서 Windows Holographic for Business 장치에 대한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
