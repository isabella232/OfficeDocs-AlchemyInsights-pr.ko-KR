# <a name="cant-create-a-team-in-teams"></a>Teams 에서 팀을 만들 수 없습니다.

Teams 관리자가 팀을 만드는 데 필요한 권한을 부여하지 않았을 수 있습니다.  

Microsoft 365에서 조직의 보안 그룹 설정을 만들거나 변경하는 방법에 대해서는 관리자에게 문의합니다. 보안 그룹에 추가된 경우 팀을 만들 수 있습니다.
