# <a name="cant-preview-all-of-an-attachment"></a>첨부 파일을 모두 미리 볼 수 없습니다.

일부 고객은 첨부 파일을 모두 미리 볼 수 없는 문제를 알고 있습니다. 이 경우 첨부 파일을 새 창에서 열면 됩니다.
