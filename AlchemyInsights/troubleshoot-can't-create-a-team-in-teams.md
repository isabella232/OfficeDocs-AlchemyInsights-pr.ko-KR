# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>팀에서 팀을 만들 수 없는 경우 Microsoft Teams

관리자에게 Teams 요청하여 다음을 할 수 있습니다.

1. 조직에서 조직의 보안 그룹 설정을 만들거나 Microsoft 365.
2. 해당 보안 그룹에 추가합니다.
