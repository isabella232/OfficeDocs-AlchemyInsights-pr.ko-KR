# <a name="user-mailbox-creation-is-taking-a-long-time"></a>사용자 사서함 만들기 시간이 오래 걸리는 경우

새 사서함을 사용할 준비가 되 면 최대 24 시간이 걸릴 수 있습니다.
  
사용자의 사서함이 1 시간 이상 "준비 중" 상태인 경우 사용자에 게 Exchange 라이선스를 제거 하 고 다시 추가 하거나 [지원 서비스에 문의](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products?tabs=online)하세요.
