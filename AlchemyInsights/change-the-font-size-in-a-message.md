# <a name="change-the-font-size-in-a-message"></a>메시지의 글꼴 크기 변경

1. 메시지를 이미 입력한 경우 텍스트를 선택합니다.
2. 메시지  **아래의 서식** 아이콘(첫 번째 아이콘)을 선택합니다.
3. 메시지 위에  **있는 글꼴**  크기 아이콘(일곱 번째 아이콘)을 선택합니다.
4. 크기를 선택합니다.
