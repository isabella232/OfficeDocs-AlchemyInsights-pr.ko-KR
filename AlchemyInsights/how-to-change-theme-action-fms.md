# <a name="how-to-change-your-theme"></a>테마를 변경 하는 방법

Outlook.com 또는 웹용 Outlook에서 테마를 변경 하려면 **설정** 으로 이동 하 여 **테마 아래에서 테마를**선택 합니다. 사용 가능한 모든 테마를 보려면 **테마**아래에서 **모두 보기** 를 선택 합니다.
