# <a name="how-to-change-your-theme"></a>테마를 변경하는 방법

Outlook.com 또는 웹용 Outlook 변경하기 위해 설정 테마 에서  테마를 **선택 합니다.** 사용 가능한 모든 테마를 표시하려면 테마 에서 **모두** 보기를 **선택합니다.**
