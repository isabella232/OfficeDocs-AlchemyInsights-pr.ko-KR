# <a name="update-dns-records"></a>DNS 레코드 업데이트

사용자 지정 도메인 이름으로 구독이 작동하도록 하려면 DNS 레코드를 설정해야 합니다. 아래 링크를 사용하여 DNS 레코드를 추가하거나 업데이트합니다.
  
- [DNS 호스팅 공급자에 상관없이 DNS 레코드 만들기](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)  
- [사용자 지정 DNS 레코드 추가 또는 편집](https://docs.microsoft.com/microsoft-365/admin/dns/add-or-edit-custom-dns-records)
