# <a name="block-pop-up-windows-in-microsoft-edge"></a>Windows에서 팝업 창 Microsoft Edge

1. 으로 **이동... (설정 등)**  >  **설정**  >  **사이트 사용 권한**.
2. 팝업 **및 리디렉션을 선택합니다.**
3. 차단을 **켜고 를 합니다.**
