# <a name="join-a-team-by-code"></a>코드로 팀 참가

Microsoft는 사용자가 **팀 가입** 코드를 만들거나 **코드별 가입** 기능을 사용하여 팀에 가입할 수 없는 문제를 알고 있습니다. 해결 방법으로 팀 소유자는 필요에 따라 사용자를 수동으로 추가할 수 있습니다.
