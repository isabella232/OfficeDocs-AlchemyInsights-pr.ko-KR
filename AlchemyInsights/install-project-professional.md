# <a name="install-project-professional"></a>Project Professional 설치

Project [Online의](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) 구독 버전과 Project 2019, 2016 또는 2013의 비 구독 버전을 설치하는 방법에 대한 자세한 내용은 Project 설치를 참조하세요. 

Project 2010 또는 2007의 경우 [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) 또는 Office [2007의](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)단계를 따릅니다. 

Project는 독립 실행형 응용 프로그램으로 판매됩니다. 아직 없는 경우 [Project를 구입하거나 시도해 보아야 합니다.](https://www.microsoft.com/evalcenter/evaluate-project)
