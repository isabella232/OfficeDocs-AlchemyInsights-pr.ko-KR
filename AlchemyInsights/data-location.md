# <a name="data-location"></a>데이터 위치

조직 프로필 페이지로 이동하고 데이터 위치 를 선택하여 관리 [](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) 센터에서 테넌트의 위치를 볼 **수 있습니다.**

구독하지 않은 응용 프로그램의 경우 고객 데이터가 [Microsoft 365 위치를 참조합니다.](https://docs.microsoft.com/office365/enterprise/o365-data-locations)
