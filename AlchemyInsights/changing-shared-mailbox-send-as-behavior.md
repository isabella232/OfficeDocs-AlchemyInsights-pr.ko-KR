# <a name="changing-shared-mailbox-send-as-behavior"></a>공유 사서함 다른 사람으로 보내기 동작 변경

기본적으로 공유 사서함에서 보낸 메시지는 공유 사서함의 보낸 편지함 폴더에 저장되지 않습니다. 대신, 메시지를 보낸 사람의 보낸 편지함 폴더에 저장됩니다.
  
모든 구성원이 보낸 전자 메일을 보게 하려는 경우 Microsoft 365 관리 센터에서  공유 사서함 세부 정보를 편집하고 보낸 항목 편집을 \> **선택하십시오.**
  
자세한 내용은 공유 사서함 설정 [구성을 참조하세요.](https://docs.microsoft.com/microsoft-365/admin/email/configure-a-shared-mailbox#allow-everyone-to-see-the-sent-email-the-replies)
