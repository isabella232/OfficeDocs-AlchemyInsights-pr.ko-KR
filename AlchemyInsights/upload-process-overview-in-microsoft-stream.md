# <a name="upload-process-overview-in-microsoft-stream"></a>Microsoft Stream에서 프로세스 개요 업로드

비디오를 업로드할 때마다 Microsoft Stream은 적응형 비트 속도 스트리밍을 사용하여 비디오를 스트리밍할 수 있도록 자동으로 여러 비트레이트로 인코딩합니다. Microsoft Stream은 뷰어의 사용 가능한 대역폭에 따라 재생되는 비디오의 비트와 품질을 최적화합니다.

- [Microsoft Stream에서 프로세스 개요 업로드](https://docs.microsoft.com/stream/upload-process-overview)
