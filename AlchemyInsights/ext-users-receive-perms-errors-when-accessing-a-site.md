# <a name="external-users-receive-permission-errors-when-accessing-a-site"></a>사이트에 액세스할 때 외부 사용자에게 사용 권한 오류가 표시됩니다.

공유하는 동안 오류 메시지를 표시하는 많은 시나리오가 있습니다. 

자세한 내용은 SharePoint Online 또는 [OneDrive에](https://docs.microsoft.com/sharepoint/troubleshoot/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business)액세스하는 동안 사용 권한 오류를 참조하세요.
