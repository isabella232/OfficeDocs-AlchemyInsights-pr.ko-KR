# <a name="were-here-to-help"></a>언제든지 도와드리겠습니다

아래에서 **지원팀에 문의하기**를 클릭하여Microsoft에서 티켓을 만드십시오. 지원 담당자가 문제 해결을 위해 곧 연락을 드릴 것입니다.

Microsoft 파트너에서 구독을 관리하는 경우 파트너와 협력하여 지원 티켓을 열어 주십시오.
