# <a name="check-for-app-using-camera"></a>카메라를 사용 하 여 앱 확인

1. **시작**을 선택한 다음 **설정 > 개인 정보 > 카메라** (ms-설정: 개인 정보-웹캠)를 선택 합니다.
2. 응용 프로그램 목록에서 "현재 사용 중" 인 앱을 찾은 다음 응용 프로그램을 닫습니다.
