# <a name="export-chat-history-in-teams"></a>팀에서 채팅 기록 내보내기

- 현재 사용자가 Microsoft 팀에서 직접 내보내기, 백업, 인쇄 및 보관 대화 기록을 사용할 수 없습니다. [UserVoice](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20)에서이 기능에 투표 하세요.
- 관리자 또는 eDiscovery 관리자는 Microsoft 365 준수 센터에서 [콘텐츠 검색](https://docs.microsoft.com/microsoft-365/compliance/content-search?view=o365-worldwide)  을 만들어 조직의 인스턴트 메시징 대화를 검색 한 다음 내보낼 수 있습니다. 자세한 내용은 [Export Content Search results](https://docs.microsoft.com/microsoft-365/compliance/export-search-results?view=o365-worldwide)를 참조 하세요.
