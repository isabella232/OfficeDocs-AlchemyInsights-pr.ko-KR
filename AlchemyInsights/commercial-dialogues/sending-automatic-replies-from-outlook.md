# <a name="sending-automatic-replies-from-outlook"></a>전자 메일에서 자동 Outlook

- 전자 메일 계정 유형에 따라 두 가지 방법으로 부재 중 자동 응답을 보낼 수 있습니다.
- 파일 **을** 클릭한 다음 사용자 **버전에** 따라 자동 Outlook.
