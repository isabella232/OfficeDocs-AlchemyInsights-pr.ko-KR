# <a name="buy-a-new-domain-from-microsoft"></a>Microsoft에서 새 도메인 구입

Microsoft에서 새 도메인을 구입하려면 다음 단계를 수행합니다.

1. 설치 > **도메인으로 이동하고** 도메인 **구입을 클릭합니다.** 
2. 마법사에 나타나는 설명을 따릅니다.
