# <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>사용자 설정을 사용하여 일정 공유를 Microsoft 365 관리 센터

1. 관리 센터에서 에서 설정   >   **또는 설정.**
2. 서비스 **탭에서** 일정 을 **선택합니다.**
3. 열 **수** 있는 일정 페이지에서 사용자가 조직 외부의 사용자와 일정을 공유할 수 있도록 할지 또는 조직 외부의 사용자와 일정을 Microsoft 365 Exchange.
4. 익명 사용자(로그온 자격 증명이 없는 사용자)가 전자 메일 초대를 통해 일정에 액세스할 수 있도록 허용할지 여부를 선택하십시오.
5. 사용자가 사용할 수 있도록 할 일정 정보 유형을 선택하십시오. 모든 정보를 허용하거나 시간만 또는 시간, 제목 및 위치로만 제한할 수 있습니다.

추가 정보: [외부 사용자와 일정 공유](https://docs.microsoft.com/microsoft-365/admin/manage/share-calendars-with-external-users)
