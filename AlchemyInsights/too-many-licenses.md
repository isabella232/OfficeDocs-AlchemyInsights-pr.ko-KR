# <a name="too-many-licenses"></a>라이선스가 너무 많나요?

너무 많은 라이선스에 대한 대금이 청구된다고 생각되는 경우 일반적으로 라이선스를 더 구입한 것이기 때문에 필요합니다.
  
**구매한 라이선스 수 줄이기**

**이 작업을 하는** 방법을 보여주는 비디오 시청: 기존 라이선스 [제거](https://go.microsoft.com/fwlink/p/?linkid=2154938)
  
1. 관리 센터에서 **청구** \> **[제품](https://go.microsoft.com/fwlink/p/?linkid=842054)** 페이지로 이동하세요.
2. **제품** 탭에서 변경할 구독을 선택합니다.
3. 구독 세부 정보 페이지에서 라이선스 **제거를 선택합니다.**
4. 라이선스 **제거 창의** 총  라이선스 수량  아래에 이 구독에 대해 원하는 총 라이선스 수를 입력합니다. 예를 들어 라이선스가 100개인 경우 25개 라이선스를 제거하려면 75를 입력합니다.
5. **저장** 을 선택합니다.

자세한 내용은 라이선스 구입 [또는 제거를 참조합니다.](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)
