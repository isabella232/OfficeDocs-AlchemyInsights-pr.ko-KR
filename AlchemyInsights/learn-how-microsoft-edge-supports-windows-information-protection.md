# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>WIP(Microsoft Edge 보호)를 Windows 방법에 대해 자세히 알아보시다.

원격 작업자 수가 증가하면 회사 장치에서 발생하는 개인 및 작업 활동의 수도 증가합니다. 그러면 회사 데이터가 회사 외부에서 공유될 위험이 높아집니다. 이러한 위험을 완화하기 위해 Microsoft Edge(버전 81 이상)는 권한이 없는 실수로 공개되지 않도록 엔터프라이즈 데이터를 보호하는 Windows-10 기능인 WIP(Windows Information Protection)를 지원하고 있습니다.
