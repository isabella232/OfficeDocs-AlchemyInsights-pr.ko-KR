# <a name="content-type-publishing"></a>콘텐츠 형식 게시

모든 사용자에 대해 콘텐츠 형식을 조직 전체에서 사용할 수 있도록 하려면 모든 사이트 모음에 게시 하는 것이 좋습니다. 콘텐츠 형식을 게시 하려면 [콘텐츠 게시 허브에서 콘텐츠 형식 게시](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02)를 참조 하세요.
