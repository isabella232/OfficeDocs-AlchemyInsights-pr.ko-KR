# <a name="cant-select-contacts-when-composing-email"></a>전자 메일을 작성하는 경우 연락처를 선택할 수 없습니다.

새 전자 메일을 작성하는 경우 일부 고객은 To 또는 **Cc**  단추를 클릭할 때 연락처 목록을 볼 수 없습니다. 연락처를 선택하려면 연락처 또는Cc  필드에 입력을 시작하면 관련 연락처가 표시됩니다.
