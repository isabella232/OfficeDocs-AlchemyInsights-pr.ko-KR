# <a name="how-to-type-special-characters-on-a-mac"></a>Mac에서 특수 문자를 입력하는 방법

Mac에서 특수 문자를 입력하기  위해 명령 제어 공간의 바로 **가기** 키로 특수 문자 메뉴를 ⌘  +  **를 니다.**  +  

@에 대한 빠른 바로 가기도 **ALT**  +  **64 입니다.**
