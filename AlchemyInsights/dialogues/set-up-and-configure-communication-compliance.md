# <a name="set-up-and-configure-communication-compliance"></a>통신 규정 준수 설정 및 구성

향후 참조를 위해 통신 규정 준수로 모니터링되는 사용자에게 Microsoft 365 E5 준수 라이선스 또는 Office 365 Enterprise E3 라이선스가 있어야 합니다.

* 통신 규정 준수를 설정하거나 구성하려는 경우 통신 규정 준수 [시작을 참조합니다.](https://go.microsoft.com/fwlink/?linkid=2111549)
* 기존 Enterprise E5 요금제가 없는 경우 감독을 사용하려는 경우 [Office 365 Enterprise E5 평가판에 등록합니다.](https://go.microsoft.com/fwlink/p/?LinkID=698279)
