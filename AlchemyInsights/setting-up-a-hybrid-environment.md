# <a name="setting-up-a-hybrid-environment"></a>하이브리드 환경 설정

데이터가 온-프레미스 서버 또는 Microsoft 365에 저장 되어 있는 경우에도 사용자가 원활 하 게 공동 작업을 수행 하도록 하려면 하이브리드 환경을 설정 해야 할 수 있습니다. 옵션에 대 한 개요를 보려면 [온-프레미스 환경과의 Microsoft 365 통합](https://docs.microsoft.com/office365/enterprise/office-365-integration)을 참조 하세요.
