# <a name="enable-versioning-for-a-list-or-library"></a>목록 또는 라이브러리에 대한 버전 설정

1. 목록 또는 라이브러리로 이동하십시오.
    
2. 를 **설정** 를 **클릭한** 다음 라이브러리 설정 또는 **목록 설정을 클릭합니다.**
    
3. 버전 **설정 을 클릭합니다.**
    
4. 원하는 설정을 선택한 다음 확인 을 **클릭합니다.**
